解决ORA-21561 OID generation failed

在Mac 下使用Navicat连接Oracle的时候会出现 提示ORA-21561: OID generation failed

解决方法

给127.0.0.1 添加对应的主机名。

通过终端查看当前主机名

hostname

然后修改hosts文件对应的映射 /etc/hosts

127.0.0.1 localhost $hostname

$hostname 修改对对应的主机名
